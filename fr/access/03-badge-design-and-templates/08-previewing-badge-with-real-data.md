Le sélecteur **APERÇU COMME** du Badge Designer vous permet de prévisualiser le badge avec les données réelles d'un employé, afin de vous assurer que le design fonctionne correctement avec vos vraies photos, noms, rôles et données personnalisées.

![Badge Designer — sélecteur APERÇU COMME dans la barre d'outils du panneau central](../screenshots/badge-designer.png)

---

## Utiliser l'aperçu avec de vraies données

1. Ouvrez le **Badge Designer** depuis la barre latérale
2. Dans la **barre d'outils du panneau central**, repérez la pastille **APERÇU COMME**
3. Cliquez dessus pour ouvrir une liste déroulante avec un champ de recherche
4. Saisissez un nom ou faites défiler pour trouver un employé
5. Cliquez sur un nom — le badge se met à jour instantanément avec les données de cet employé

---

## L'employé simulé par défaut : John Doe

Même si vous n'avez encore ajouté aucun employé réel, le Badge Designer affiche toujours un aperçu avec l'employé simulé **John Doe** :

- **Nom** : John Doe
- **ID Employé** : EMP-001
- **Rôle** : Ingénieur logiciel
- **Département** : Ingénierie

John Doe est toujours disponible dans le sélecteur et est utile pour évaluer les mises en page avant que des données réelles ne soient saisies.

---

## Ce que le sélecteur remplit

Lorsque vous sélectionnez un empleoyé réel, ces données remplissent le badge :

- Prénom et nom
- Photo (si téléchargée)
- Rôle
- Département
- ID Employé
- ID Badge
- Champs personnalisés ayant l'option "Afficher sur le badge" activée

---

## Pourquoi prévisualiser avec de vraies données

L'aperçu avec de vraies données vous permet de vérifier :

- **Troncature du texte** — Les noms longs s'affichent-ils correctement ou sont-ils coupés ?
- **Qualité de la photo** — La photo de l'employé s'insère-t-elle bien dans le cadre de la mise en page ?
- **Champs personnalisés** — Les valeurs longues s'adaptent-elles sans déborder ?
- **Contenu vide** — Les éléments se réorganisent-ils correctement quand certains champs sont vides ?

Testez avec plusieurs employés ayant des profils variés pour garantir la cohérence sur l'ensemble du jeu de données.

---

## L'aperçu n'est pas sauvegardé

La sélection dans le sélecteur APERÇU COMME est temporaire et ne sauvegarde rien. Elle affecte uniquement ce que vous voyez dans le panneau central. Pour appliquer et enregistrer les modifications du design, cliquez sur **Enregistrer** dans le panneau droit.
