La fonctionnalité Suggestions de thème par IA génère automatiquement un thème de badge complet à partir d'une description en langage naturel. Décrivez le style que vous souhaitez et l'IA configure la mise en page, la police, les couleurs et les éléments en un seul clic.

![Badge Designer — section Thème IA dans le panneau droit](../screenshots/badge-designer.png)

---

## Accéder aux suggestions de thème IA

1. Ouvrez le **Badge Designer** depuis la barre latérale
2. Dans le **panneau droit**, assurez-vous d'être dans l'onglet **Marque**
3. Trouvez la section **Thème IA** (disponibilité selon votre abonnement)

> Si vous ne voyez pas la section Thème IA, cette fonctionnalité n'est pas incluse dans votre abonnement actuel. Contactez votre responsable de compte pour en savoir plus.

---

## Générer un thème

1. Saisissez une description du style souhaité dans le champ de texte
2. Cliquez sur **Générer le thème**
3. L'IA configure automatiquement :
   - La **mise en page** (parmi les 7 disponibles)
   - La **police** (sans serif, serif ou monospace)
   - La **couleur principale** et la **couleur d'accentuation**
   - Les **éléments du recto** (quels champs sont affichés)
   - Le **nom de l'entreprise** et le **slogan** si pertinent

---

## Conseils pour de meilleurs résultats

Soyez précis dans votre description pour obtenir de meilleurs thèmes :

| Vague | Précis |
|---|---|
| "Professionnel" | "Cabinet juridique formel avec couleurs bleu marine et or, style corporate classique" |
| "Moderne" | "Startup technologique, palette sombre avec accents verts, minimaliste" |
| "Créatif" | "Agence créative, tons violets vibrants, mise en page dégradé, sans serif moderne" |

Itérez avec différentes descriptions si le premier résultat ne correspond pas — chaque génération peut produire un résultat différent.

---

## Ce que l'IA ne peut pas faire

La génération de thème IA **ne peut pas** :

- Définir ou remplacer votre logo (à faire manuellement dans l'onglet Marque)
- Configurer le verso du badge
- Enregistrer automatiquement — cliquez toujours sur **Enregistrer** après avoir approuvé un thème généré

---

## Modifier après génération

Tous les paramètres définis par l'IA peuvent être ajustés manuellement. Utilisez le résultat de l'IA comme point de départ et affinez les couleurs, la mise en page ou les éléments selon vos préférences.
