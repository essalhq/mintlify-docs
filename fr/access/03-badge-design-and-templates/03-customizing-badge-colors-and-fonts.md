L'onglet Marque du panneau droit du Badge Designer vous permet de définir les couleurs, la police, le logo et l'image de fond qui s'appliquent à l'ensemble du modèle de badge.

![Badge Designer — panneau Marque](../screenshots/badge-designer.png)

---

## Accéder aux paramètres de marque

1. Ouvrez le **Badge Designer** depuis la barre latérale
2. Dans le **panneau droit**, assurez-vous d'être dans l'onglet **Marque**

---

## Couleur principale

La couleur principale est utilisée pour les **barres d'en-tête, les puces de rôle et les bandes d'accentuation**.

- Cliquez sur la pastille de couleur pour ouvrir le sélecteur de couleur
- Ou saisissez directement un code hexadécimal (ex. : `#1A4F9D`)

---

## Couleur d'accentuation

La couleur d'accentuation est utilisée pour les **bandes de pied de page, les arrêts de dégradé et la couleur de fond du verso du badge**.

- Idéalement une couleur complémentaire ou secondaire de votre identité visuelle
- Cliquez sur la pastille ou saisissez un code hexadécimal

---

## Logo

Téléchargez le logo de votre organisation. Le logo apparaît sur le **recto et/ou le verso** du badge selon ce qui est activé dans la section Éléments.

- Formats acceptés : PNG, SVG, JPG, WEBP
- Si aucun logo n'est téléchargé, le nom de l'entreprise s'affiche en texte dans la couleur principale

---

## Image de fond (Recto)

Téléchargez une image pour la rendre en arrière-plan de tous les éléments du badge sur le recto.

- Taille maximale : 2 Mo
- L'image est rendue derrière tout le contenu — veillez à ce qu'elle n'affecte pas la lisibilité du texte

---

## Police

Change la police utilisée pour **l'ensemble du texte du badge**.

| Option | Apparence |
|---|---|
| **Sans serif** (défaut) | Moderne et propre (ex. : Inter, Helvetica) |
| **Serif** | Classique et traditionnel (ex. : Times New Roman) |
| **Monospace** | Style technique (ex. : Courier) |

> La police est l'un des rares éléments que vous ne pouvez changer qu'ici ou via les suggestions de thème IA — il n'existe pas d'autre moyen de la modifier.

---

## Enregistrer les modifications

Cliquez sur **Enregistrer** en haut à droite du panneau droit. Les modifications s'appliquent immédiatement à tous les badges.
