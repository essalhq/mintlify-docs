La section Éléments du panneau gauche vous permet de choisir quelles informations s'affichent sur le recto du badge. Vous pouvez activer ou désactiver chaque champ indépendamment.

![Badge Designer — recto avec éléments actifs](../screenshots/badge-designer.png)

---

## Accéder à la configuration du recto

1. Ouvrez le **Badge Designer** depuis la barre latérale
2. Assurez-vous que le sélecteur **Recto/Verso** dans le panneau central est réglé sur **Recto**
3. Dans le **panneau gauche**, ouvrez la section **Éléments**

---

## Les 9 champs activables

Chaque champ peut être activé ou désactivé individuellement via son bouton bascule :

| Champ | Description |
|---|---|
| **Photo** | La photo de l'employé (circulaire ou carrée selon la mise en page) |
| **Rôle** | Le titre de poste de l'employé |
| **Département** | Le département auquel l'employé est affecté |
| **ID Employé** | L'identifiant interne de l'employé (ex. : EMP-001) |
| **ID Badge** | L'identifiant unique du badge, utilisé pour le scan |
| **Groupe sanguin** | Affiché si renseigné dans la fiche employé |
| **QR Code** | Renvoie vers `idpage.link/public/{idBadge}` — la page de profil public de l'employé |
| **Slogan** | Courte phrase définie dans les paramètres de l'organisation |
| **Logo** | Le logo de l'entreprise (utilise l'image téléchargée dans les paramètres de marque) |

---

## Étiquettes de titre

Pour ces champs, un sous-bouton **Afficher l'étiquette de titre** est disponible :
- Rôle
- Département
- ID Employé
- ID Badge
- Groupe sanguin

Lorsqu'il est activé, le libellé du champ s'affiche au-dessus de la valeur (ex. : "Département : Ingénierie"). Désactivez-le pour n'afficher que la valeur.

---

## Champs toujours affichés

Le **Prénom** et le **Nom** de l'employé sont toujours affichés sur le badge. Il n'est pas possible de les masquer.

---

## Champs personnalisés

Les champs personnalisés configurés dans Paramètres → Champs personnalisés et ayant l'option **Afficher sur le badge** activée apparaissent également dans la section Éléments. Activez ou désactivez-les de la même façon que les champs standards.

---

## Enregistrer les modifications

Cliquez sur **Enregistrer** en haut à droite du panneau droit. Les modifications s'appliquent à tous les badges immédiatement.
