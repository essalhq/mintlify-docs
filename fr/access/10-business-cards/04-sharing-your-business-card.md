---
title: "Partager Votre Carte de Visite"
---
{/* category: Business Cards */}


La carte de visite numérique d'un employé peut être partagée de plusieurs manières : via un lien direct, un code QR ou un fichier vCard téléchargeable. Aucune application ou compte n'est requis pour le destinataire.

![Page d'accueil du Portail Employé](../screenshots/portal-home.png)

---

## Partage via un Lien

Chaque employé possède une URL publique basée sur son ID de badge :

```
https://{domaine-de-votre-entreprise}/{badgeId}
```

Copiez ce lien et partagez-le par e-mail, application de messagerie ou tout autre canal. Lorsque le destinataire l'ouvre, il voit directement votre carte de visite — aucune connexion n'est requise.

Vous pouvez trouver votre ID de badge sur la page **Mon Profil** dans le Portail Employé, dans le champ **ID Badge**.

---

## Partage via le Code QR

Le code QR de votre badge renvoie directement à votre carte de visite (ou à votre profil public complet, selon les paramètres de votre organisation).

**Pour afficher votre code QR :**

1. Connectez-vous au Portail Employé.
2. Allez sur **Accueil**.
3. Cliquez sur **Afficher le Code QR**.
4. Un code QR plein écran s'affiche — laissez l'autre personne le scanner avec son téléphone.

Les **badges physiques** contiennent également un code QR. Scanner le badge imprimé avec n'importe quel appareil photo de téléphone ouvre automatiquement la page publique.

---

## Enregistrer un Contact (vCard)

Le bouton **Enregistrer le Contact** sur la carte de visite génère un fichier vCard (`.vcf`) et le télécharge sur l'appareil du visiteur. La vCard inclut :

- Le nom complet et l'intitulé du poste
- L'organisation / département
- Le numéro de téléphone et l'adresse e-mail
- L'adresse (si l'emplacement est affiché)
- Le site web et les liens personnalisés
- Les liens vers les profils de réseaux sociaux

La plupart des téléphones et clients de messagerie peuvent importer directement les fichiers `.vcf` dans leur application de contacts.

---

## Ce que le destinataire voit

Le destinataire ouvre une page adaptée aux mobiles affichant :

- Votre couverture aux couleurs de l'entreprise.
- Votre photo, nom, poste et département.
- Des liens icônes vers vos réseaux sociaux.
- Le bouton Enregistrer le Contact et des boutons d'action rapide (E-mail, Appel).
- Votre site web et vos liens personnalisés.
- Votre biographie et vos coordonnées.

Aucun compte, application ou connexion n'est requis du côté du destinataire.
