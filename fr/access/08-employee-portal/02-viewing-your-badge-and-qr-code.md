La page d'**Accueil** est la première chose que vous voyez après vous être connecté. Elle affiche un aperçu 3D interactif de votre badge actuel et vous donne un accès rapide à votre code QR.

![Page d'accueil du Portail Employé](../screenshots/portal-home.png)

---

## Votre Aperçu de Badge en 3D

Votre badge est affiché sous la forme d'une carte 3D interactive. Vous pouvez l'incliner et la faire pivoter en déplaçant votre souris dessus (ou en inclinant votre appareil si vous êtes sur un mobile). Appuyez ou cliquez sur le badge pour le retourner et voir le verso.

Le badge reflète votre design actuel et vos informations personnelles telles que configurées par votre administrateur.

---

## Afficher Votre Code QR

1. Cliquez sur **Afficher le Code QR** sur la page d'accueil.
2. Un code QR s'affiche en plein écran.
3. Laissez quelqu'un le scanner, ou scannez-le avec un appareil d'enregistrement, pour vérifier votre badge.
4. Cliquez sur **Terminé** pour fermer.

Le code QR renvoie à votre page de vérification de badge et est toujours à jour.

---

## Actions Rapides

La page d'accueil propose également des raccourcis :

- **Mon Profil** — accédez directement à la page d'édition de votre profil.
- **Afficher le Code QR** — ouvrez le code QR en plein écran.
- **Signaler un Problème** — rendez-vous sur la page de signalement de badge manquant.

---

## Si votre badge ne s'affiche pas

Si la page d'accueil affiche un état vide ou aucun aperçu de badge, il est possible que votre badge n'ait pas encore été attribué ou qu'il ait été désactivé. Contactez votre service RH ou votre administrateur informatique pour qu'un badge soit créé et associé à votre compte.
